Commands to make a project and start tracking history

  • cd Desktop
  • mkdir my-project
  • cd my-project
  • touch index.html
  • code .
  • git init
  • git add index.html
  • git commit -m "my message"
  • git log
  • (exit with :q)

Steps to publish our code on github

  1. Make a repository on github
  2. Add a remote
  3. git remote add origin ...url (NOT HTTPS!!111!)
  4. git push origin master
