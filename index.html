Commands to make a project and start tracking history

  • cd Desktop
  • mkdir my-project
  • cd my-project
  • touch index.html
  • code .
  • git init
  • git add index.html
  • git commit -m "my message"
  • git log
  • (exit with :q)

Steps to publish our code on github

  1. Make a repository on github
  2. Add a remote
  3. git remote add origin ...url (NOT HTTPS!!111!)
  4. git push origin master

Steps to make a branch and update the master

  1. Create a branch
  2. git checkout -b feature/myfeaturename
  3. Write some code
  4. Commit your changes (see instruction above)
  5. Push to the remote to a new branch
  6. git push origin feature/myfeaturename
  7. Go to github
  8. Go to your project on github
  9. Make a "Pullrequest" to master from the branch you pushed
  10. Approve pullrequest
  11. On you local git, switch back to master
  12. git checkout master
  13. Pull the changes from the remote
  14. git pull origin master
